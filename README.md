## NumGame 支援這些裝置。

* 建議 **4 GB RAM** 以上（**不是必須**）
* 建議**剩餘1.5 GB的磁碟空間**（**不是必須**）
* 必須**運行 macOS**，但**不一定需要是Mac**。
